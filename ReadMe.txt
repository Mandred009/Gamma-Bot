GAMMA BOT

Mandred Industries' Robotvis Mini personal desktop assistant

1) Can open applications whose path is preprogrammed.

2) Inactivity timer which alerts the user to move after 1 hour. ( Can be changed).

3) Can show the weather stats of any city in the world real time.

4) Can display time and toggle 6 different rgb led combinations on command. 

5) Can search browser and youtube as well and changes its icon ie on its face.( by default 3 browsers - chrome, brave, edge).

6) Can open or close its arms on command inorder to hold a pen or any suitable object.
